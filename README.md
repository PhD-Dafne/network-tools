# network-tools
nwtools is a collection of network analysis tools on top of NetworkX and iGraph.

See the [documentation](https://network-tools.readthedocs.io/en/latest/).

## Installation
Prerequisites:
- Python 3.5 or 3.6

Install the module with:
```
pip install .
```
